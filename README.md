# airhornbot

A Node.js implementation of Airhorn Bot.

# Setup

Prerequisites:
- Yarn

## Bot

Make sure to update/create `config.json`!

Example commands:
```
cd bot
yarn install
yarn run build
```

To register the slash commands:
```
cd bot
yarn run register-commands
```

To run the bot:
```
cd bot
yarn run bot
```
